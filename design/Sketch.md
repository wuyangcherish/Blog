# Sketch

## 用法总结

### 快捷键

1. **R** : 画出一个长方形
2. **U** : 画出一个圆角为1 的圆角矩形
3. **L** : 画出一条直线
3. **command+D** : 复制
4. 按住 **enter** 然后就进入了编辑模式，如果想改变形状等，可以添加节点，然后修改成自己想要的样式，比 **PS** 里面的好用点~

### 工具

想找到某个工具但是不知道在哪里就在 **Help** 中搜搜

Tools 里面有很多的工具，点击找到**Tools** 然后在里面查找到自己要用的，可以拖动到程勇的工具栏
